Trabalhando com listas

Listas ordenadas

  1. Acordar
  2. Ligar para o jão
  3. Tomar café
  4. Escovar os dentes

Listão são criadas fora do paragrafo <p>, e são ordenadas pela tag <ol>(ordered list) da seguinte forma 

 <ol>
            <li>Acordar</li>
            <li>Ligar para o Jão</li>
            <li>Tomar café</li>
            <li>Escovar os dentes</li>
Lembrando que as listas não precisam do fechamento </li> e nesse modelo em especifico não existe uma ordenação pre definida.

Ordenação de listas

Para escolhermos a ordenação de listas precisamos especificar o tipo de ordenação usando o atibuto type da seguinte forma 

 <ol type="A">
            <li>Acordar</li>
            <li>Ligar para o Jão</li>
            <li>Tomar café</li>
            <li>Escovar os dentes</li>
        </ol>
Que ordanaria da seguinte maneira:
  1. Acordar
  2. Ligar para o jão
  3. Tomar café
  4. Escovar os dentes

Ou

 <ol type="1" start="2">
            <li>Acordar</li>
            <li>Ligar para o Jão</li>
            <li>Tomar café</li>
            <li>Escovar os dentes</li>
        </ol>
Que ficaria:
  1. Acordar
  2. Ligar para o jão
  3. Tomar café
  4. Escovar os dentes

Listas não ordenadas

Listas não ordenadas São aquelas que não trazem uma ordem específica <ul>(unordered list) 

 <ul>
            <li>Acordar</li>
            <li>Ligar para o Jão</li>
            <li>Tomar café</li>
            <li>Escovar os dentes</li>
        </ul>
Que se ordenam da seguinte forma
  • Acordar
  • Ligar para o jão
  • Tomar café
  • Escovar os dentes

Juntando listas

Para criar lisjas juntas ou listas com listas basta adicionar uma tag <ol> dentro de um elemento <li> da seguinte forma:

  1. Assassin's Creed
    • Assassin's Creed 1
    • Assassin's Creed 2
    • Assassin's Creed Brotherhood
  2. God of War
    • God of War I
    • God of War II
    • God of War III
  3. GTA

Link externos

Quando se deseja fazer a navegação esterna para uma outra pagina ou site e necessario utilizar um link externo. e emcapsular ele em uma fraze ou texto de preferencia sua extrutura sera da seguinte forma: <a href="link">link para uma pagina X</a>

Exemplo: Google

Aprofundando mais sobre os links

Notado que o link a cima não te redirecionou para uma pagina especifica, mas sim sobrepoem a pagina atual, para isso iremos acrescentar um atributo target para redirencionar a pagina a estrutura ficaria da seguinte forma: <a href="link" target="tipo de redirecionamento">link para uma pagina X</a>

  1. _blank = Vai abrir uma janela em branco
    • Exemplo: Google
  2. _self = Vai abrir o link na janela ou frame atual (padrão)
    • Exemplo: Google
  3. _top = Vai desfazer todos os frames e abrir o destino no navegador completo
    • Exemplo: Google
  4. _parent = Similar ao uso do _top em uma referência a janela pai
    • Exemplo: Google

Link internos

Os links internos são utilizados para fazer navegação entre paginas do mesmo servidor ou no caso desse projeto na mesma pasta do arquivo. EX:

Paginas 1, 2, 3
