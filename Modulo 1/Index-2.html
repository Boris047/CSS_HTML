Podemos criar também Um texto marcado utilizando a tag <Mark>

Podemos criar também Um texto marcado utilizando a tag <Mark> e um css para estilo, acrescentando um style dento do "parentese" da seguinte forma <mark style="background-color:blue"

MOMENTO LEMBRETE!!!!!!

Caracteristica de style não sao erdadas ou seja o style criado dento de uma linha especifica do codigo NÃO SÃO ERDADAS PARA AS PROXIMAS LINHAS!!!!! para isso utilizamos o espaço head no inicio do codigo para definir estilos.
EX: 

    <head>
        <style>
            mark {
                background-color: blue;
                }
        <style>
